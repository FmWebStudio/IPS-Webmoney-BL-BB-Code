Описание

ББ Код Webmoney BL позволяет отобразить ваш BL в сообщении.

Шаг 1: Залить файлы

Залейте содержимое папки upload в корень форума.

Шаг 2: Импортируйте файл медиа-тега

Войдите в АЦ и перейдите к Внешний вид -> Сообщения -> ББ Коды . Внизу страницы увидите блок "Импортировать новые BB-коды", там выберите XML файл бб кода и нажмите Импортировать.
